# Say hello to someone

You can say hello to someone in particular using `say_hello()`.

``` r
say_hello() # Hello Berra
#> Hello Berra
#> Hello Berra
say_hello(someone = "Seb") # Hello Seb
#> Hello Seb
#> Hello Seb

hello_anna <- say_hello(someone = "Anna") 
#> Hello Anna
hello_anna
#> Hello Anna
```
